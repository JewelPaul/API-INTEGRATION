About Us

Welcome to Global Pulse News, a dynamic news platform designed to bring you the latest updates from around the world.

This website was created by Jewel Gabriel Paul as part of a project/task assigned by CODTECH. The goal of this project was to develop a fully functional news website that fetches real-time news articles using an API, with a focus on responsive design, user-friendly navigation, and professional presentation.

About the API

The news articles displayed on this website are sourced from The Guardian API. The Guardian API provides access to a vast collection of news articles, allowing developers to fetch and display content in various categories such as World, Business, Entertainment, and more. For more information about The Guardian API, visit their official documentation at https://open-platform.theguardian.com/.

Project Details

Global Pulse News features a responsive design that works seamlessly on both mobile and desktop devices. Key features include:

  • A breaking news ticker for real-time updates.
  • Category-based navigation (e.g., World, India, Tech).
  • Sidebars for trending news (Spotlight) and entertainment (World of Stars).
  • Professional pagination with clickable page numbers to jump directly to specific pages.
  • City-based filtering to view news from specific regions (e.g., India, USA, UK).

This project was an opportunity to learn and apply skills in API integration, responsive web design, and JavaScript programming. Thank you for visiting Global Pulse News!
