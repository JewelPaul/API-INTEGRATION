Privacy Policy

Global Pulse News values your privacy. This website was created by Jewel Gabriel Paul as part of a project/task assigned by CODTECH. This privacy policy outlines how we handle data on this website.

1. Data Collection

Global Pulse News is a student project and does not collect any personal information from users. The website does not use cookies, forms, or any other means to gather user data.

2. API Data Usage

The news articles displayed on this website are sourced from The Guardian API. The Guardian API provides access to news content, which is fetched and displayed on this website. No user data is sent to The Guardian API, and the API is used solely to retrieve publicly available news articles. For more information about The Guardian API and its data policies, please visit https://open-platform.theguardian.com/.

3. Third-Party Links

This website contains links to external sites, such as news articles from The Guardian and The Guardian API documentation. We are not responsible for the privacy practices of these third-party websites.

4. Changes to Privacy Policy

This privacy policy may be updated as needed. Since this is a student project, changes will be minimal and primarily for educational purposes.

For any questions about this privacy policy, please refer to the Contact page.
